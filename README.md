# MyBB-中文语言包

为 MyBB 论坛系统提供的简体中文语言包。本语言包基于 MyBB 1.8.39 官方英文语言包翻译

欢迎提交翻译修正、新术语建议或报告问题

将语言包里的内容解压到 inc\languages 

在管理员后台

 - 设置控制面板语言: Home - Preferences - Admin Control Panel Language - 选择中文(简体) 并保存设置

 - 设置论坛默认语言: Configuration - Settings - General Configuration - Default Language - 选择中文(简体) 并保存设置
